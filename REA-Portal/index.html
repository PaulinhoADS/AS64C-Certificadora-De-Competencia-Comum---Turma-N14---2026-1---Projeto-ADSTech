Desvende o mundo do Front-End

Você já se perguntou como os sites que você usa todos os dias são construídos? O Front-End é a parte visual da web, tudo aquilo com o qual você interage: botões, cores, textos e imagens. Neste mini-curso, vamos te ensinar a construir a sua própria loja virtual (e-commerce) do zero, de forma simples e direta.

Trilha de Conhecimento

Nossos módulos foram pensados para quem nunca programou. Siga a ordem e construa sua loja!

1. Estrutura (HTML)

O esqueleto do site. Aprenda a colocar os textos, títulos e imagens na tela.

2. Estilo (CSS)

A maquiagem do site. Descubra como aplicar cores, fontes e organizar o layout.

3. Ação (JavaScript)

O cérebro do site. Crie interatividade, como o botão de adicionar ao carrinho.

Sobre o Projeto E-commerce Front-End

Este portal é o resultado prático de uma jornada de aprendizado contínuo. Nosso objetivo com os módulos de ensino é desmistificar a programação para leigos, mostrando que qualquer pessoa pode construir uma interface web do zero.

Ao passar pelas aulas de HTML, CSS e JavaScript, o aluno constrói, passo a passo, uma vitrine virtual funcional. O Front-End moderno vai além de códigos; trata-se de criar experiências, acessibilidade e interfaces responsivas. Este projeto comprova que a união de metodologias ágeis com o desenvolvimento web resulta em soluções reais e prontas para o mercado de tecnologia.

A Jornada até Aqui

Nossa evolução ao longo de 4 semestres no curso de Análise e Desenvolvimento de Sistemas.

1º Semestre: A Base

Lógica de Programação, Arquitetura de Computadores, Design de Interação (IHC) e Matemática. A fundação de como os computadores operam e como pensar soluções.

2º Semestre: Dados e Sistemas

Programação Orientada a Objetos, Banco de Dados, Estrutura de Dados e Sistemas Operacionais. Criando sistemas mais robustos e armazenando informações.

3º Semestre: Web e Gestão

Desenvolvimento Web, Modelagem, Redes e Gestão de Projetos de TI (com foco em Desenvolvimento Ágil). O salto para a internet e trabalho em equipe.

4º Semestre: Aplicação (Atual)

Desenvolvimento Web Front-End e Certificadora de Competência Comum. A união de tudo aprendido para a criação deste Recurso Educacional Aberto (REA).
